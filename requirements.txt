# ---- Pika SDK (vendored under collect/pika_sdk/) ----
# The Pika SDK source lives in-tree at collect/pika_sdk and is auto-injected
# into sys.path by collect/collect_pika.py and utils/pika_interface.py — no
# pip install is required. The Vive-tracker driver (pysurvive) is the only
# extra dependency for Pika teleop and is system-specific to install:
#
#   git clone https://github.com/cntools/libsurvive.git && cd libsurvive
#   sudo apt install build-essential cmake libusb-1.0-0-dev liblapacke-dev \
#                    libopenblas-dev zlib1g-dev libatlas-base-dev
#   make && sudo make install
#   pip install pysurvive
#
# If pysurvive is not installed the Pika SDK still loads — only get_pose()
# (i.e. tracker-driven teleop) is unavailable; gripper + encoder + recording
# all keep working.
